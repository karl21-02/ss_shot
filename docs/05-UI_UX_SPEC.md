# 05. UI/UX 명세서 (UI/UX Specification)

> 화면별 상세 레이아웃 및 인터랙션 정의

---

## 1. 디자인 원칙

### 1.1 Core Principles

| 원칙 | 설명 |
| --- | --- |
| **Simple** | 핵심 기능에 집중, 불필요한 요소 배제 |
| **Fast** | 즉각적인 반응, 부드러운 애니메이션 |
| **Fun** | 게이미피케이션 요소로 지루함 해소 |

### 1.2 Color Palette

```
Primary:     #2196F3 (Blue)
Secondary:   #4CAF50 (Green)
Danger:      #F44336 (Red)
Background:  #FAFAFA (Light Gray)
Surface:     #FFFFFF (White)
Text:        #212121 (Dark Gray)
```

### 1.3 Typography

| 용도 | 스타일 |
| --- | --- |
| **Headline** | 24sp, Bold |
| **Title** | 18sp, SemiBold |
| **Body** | 14sp, Regular |
| **Caption** | 12sp, Regular |

### 3.2 컴포넌트 상세

#### Search Bar

| 속성 | 값 |
| --- | --- |
| **높이** | 48dp |
| **모서리** | 24dp (Pill Shape) |
| **아이콘** | 검색 아이콘 (왼쪽) |
| **힌트 텍스트** | "어떤 정보를 찾으세요?" |
| **동작** | 터치 시 검색 화면으로 전환 |

#### Insight Banner

| 속성 | 값 |
| --- | --- |
| **높이** | 80dp |
| **모서리** | 12dp |
| **배경** | Gradient (Primary) |
| **가로 스크롤** | PageView (여러 인사이트 카드) |

#### Category Tabs

| 속성 | 값 |
| --- | --- |
| **높이** | 48dp |
| **스타일** | Chip 형태 |
| **Sticky** | 스크롤 시 상단 고정 |
| **애니메이션** | 선택 시 배경색 전환 |

#### Gallery Grid Item

| 속성 | 값 |
| --- | --- |
| **열 수** | 3 |
| **간격** | 2dp |
| **비율** | 1:1 (정사각형) |
| **캐싱** | CachedNetworkImage 사용 |

**동기화 상태 아이콘 (우측 하단):**
| 아이콘 | 의미 | 색상 |
| --- | --- | --- |
| ☁️ | 분석 대기 (Local Only) | Gray |
| ✅ | 분석 완료 (Synced) | Blue |
| ⚠️ | 업로드 실패 | Red |

**가격 태그 (좌측 상단, 쇼핑 카테고리만):**
- 배경: 반투명 검정
- 텍스트: 흰색, 12sp

## 5. 상세 뷰어 (Detail Viewer)

### 5.1 레이아웃

```
┌─────────────────────────────────────┐
│ ←      🛍 쇼핑 ▾      ⋮ 공유 ❤ 🗑  │  ← AppBar
├─────────────────────────────────────┤
│                                     │
│                                     │
│                                     │
│         ┌─────────────┐             │
│         │  [노란색    │             │  ← 검색 하이라이트
│         │   박스]     │             │     (2초 후 Fade-out)
│         └─────────────┘             │
│                                     │
│      [스크린샷 이미지]               │  ← InteractiveViewer
│                                     │     (핀치 줌 지원)
│                                     │
│                                     │
│                                     │
├─────────────────────────────────────┤
│  ┌───────────────┐ ┌─────────────┐ │
│  │ 📄 계좌 복사   │ │ 🏦 은행 열기 │ │  ← Smart Action Bar
│  └───────────────┘ └─────────────┘ │
└─────────────────────────────────────┘
```

### 5.2 Smart Action Bar 케이스별

#### Case 1: 계좌번호 감지

```
│  ┌───────────────┐ ┌─────────────┐ │
│  │ 📄 계좌 복사   │ │ 🏦 은행 열기 │ │
│  └───────────────┘ └─────────────┘ │
```

#### Case 2: URL 감지

```
│  ┌───────────────┐ ┌─────────────┐ │
│  │ 🌐 사이트 방문 │ │ 🔗 링크 복사 │ │
│  └───────────────┘ └─────────────┘ │
```

#### Case 3: 날짜 감지

```
│  ┌───────────────┐                 │
│  │ 📅 1월 25일 캘린더 등록         │ │
│  └───────────────┘                 │
```

#### Case 4: 감지된 정보 없음

```
│                                     │
│  (Smart Action Bar 숨김)            │
│                                     │
```

## 7. 애니메이션 명세

| 화면 | 애니메이션 | Duration |
| --- | --- | --- |
| **홈 → 상세** | Hero 전환 (썸네일 → 풀이미지) | 300ms |
| **검색 하이라이트** | Fade-in → 2초 유지 → Fade-out | 500ms + 2000ms + 500ms |
| **정리 모드 스와이프** | 카드 회전 + 이동 | 200ms |
| **정리 완료** | Lottie 폭죽 | 2000ms |
| **탭 전환** | Cross-fade | 200ms |
